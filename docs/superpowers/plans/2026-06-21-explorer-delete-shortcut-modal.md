# Explorer Delete: tecla Del + modal de confirmacion con papelera — Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Borrar ficheros/carpetas en el explorer con la tecla `Del`, mostrando un modal de confirmacion con tres acciones (Cancel, Delete permanente, Move to trash).

**Architecture:** Se anade un comando Rust `fs_trash` (papelera del sistema via crate `trash`) junto al `fs_delete` permanente ya existente. En el frontend se registra el shortcut reasignable `file.delete`, se crea el componente `DeleteEntryModal` (sobre `AlertDialog` de shadcn) y se levanta el estado del modal a `FileExplorer`, compartido por la tecla Del y el menu contextual. El menu contextual deja de confirmar con doble-click.

**Tech Stack:** Rust (Tauri 2, crate `trash`), React 19 + TypeScript, shadcn `AlertDialog`, vitest, cargo test.

```rust
/// Moves a file or directory to the system trash. Unlike `fs_delete` this is
/// recoverable from the OS trash/recycle bin.
#[tauri::command]
pub fn fs_trash(path: String, workspace: Option<WorkspaceEnv>) -> Result<(), String> {
    let workspace = WorkspaceEnv::from_option(workspace);
    let p = resolve_path(&path, &workspace);
    std::fs::symlink_metadata(&p).map_err(|e| {
        log::debug!("fs_trash stat({}) failed: {e}", p.display());
        e.to_string()
    })?;
    trash::delete(&p).map_err(|e| {
        log::warn!("fs_trash({}) failed: {e}", p.display());
        e.to_string()
    })
}
```

### Task 4: Componente `DeleteEntryModal`

**Interfaces:**

- [ ] **Step 1: Create the component**

Crear `src/modules/explorer/DeleteEntryModal.tsx`:

```tsx
import { useRef } from "react";
import {
  AlertDialog,
  AlertDialogAction,
  AlertDialogCancel,
  AlertDialogContent,
  AlertDialogDescription,
  AlertDialogFooter,
  AlertDialogHeader,
  AlertDialogTitle,
} from "@/components/ui/alert-dialog";

export type DeleteEntryModalProps = {
  open: boolean;
  name: string;
  isDir: boolean;
  onCancel: () => void;
  onDelete: () => void;
  onTrash: () => void;
};

export function DeleteEntryModal({
  open,
  name,
  isDir,
  onCancel,
  onDelete,
  onTrash,
}: DeleteEntryModalProps) {
  const contentRef = useRef<HTMLDivElement>(null);
  return (
    <AlertDialog
      open={open}
      onOpenChange={(next) => {
        if (!next) onCancel();
      }}
    >
      <AlertDialogContent
        ref={contentRef}
        onOpenAutoFocus={(e) => {
          // Default focus on the safe action (Move to trash), not Delete.
          e.preventDefault();
          contentRef.current
            ?.querySelector<HTMLElement>("[data-autofocus]")
            ?.focus();
        }}
      >
        <AlertDialogHeader>
          <AlertDialogTitle>
            Delete {isDir ? "folder" : "file"} "{name}"?
          </AlertDialogTitle>
          <AlertDialogDescription>
            Move it to the system trash, or delete it permanently. Permanent
            deletion cannot be undone.
          </AlertDialogDescription>
        </AlertDialogHeader>
        <AlertDialogFooter>
          <AlertDialogCancel onClick={onCancel}>Cancel</AlertDialogCancel>
          <AlertDialogAction variant="destructive" onClick={onDelete}>
            Delete
          </AlertDialogAction>
          <AlertDialogAction data-autofocus onClick={onTrash}>
            Move to trash
          </AlertDialogAction>
        </AlertDialogFooter>
      </AlertDialogContent>
    </AlertDialog>
  );
}
```
