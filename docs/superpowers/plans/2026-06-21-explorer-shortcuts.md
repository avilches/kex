# Explorer Shortcuts & Sidebar UX Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Add 4 explorer view shortcuts (Ctrl+1-4), decouple "show panel" from "toggle panel", add a Toggle Sidebar shortcut, and switch the macOS View menu to use native checkmarks.

**Architecture:** Three independent layers touch this feature: (1) the shortcut registry (`shortcuts.ts`) declares all IDs and default bindings, (2) `App.tsx` wires the handlers and drives the macOS menu sync, (3) `FileExplorer.tsx` reads the registry to render shortcut hints in its view-mode dropdown. The Rust menu in `lib.rs` is updated independently to use `CheckMenuItem`.

**Tech Stack:** TypeScript / React 19, Tauri 2 / Rust, `tauri::menu::CheckMenuItem`

  key={m.id}
  disabled={info.disabled}
  onSelect={() => onChangeRootMode(m.id)}
  className="flex items-start gap-2.5"

  <HugeiconsIcon
    icon={m.icon}
