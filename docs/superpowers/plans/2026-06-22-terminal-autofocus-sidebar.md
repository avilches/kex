# Terminal Autofocus + sidebar-driving terminal Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Replace the explorer's "Follow Terminal" / "Follow Git" modes with a per-terminal `autofocus` flag, and make a single terminal drive both sidebar views (Explorer + Source Control / Git History) only via F4 or autofocus.

**Architecture:** A pure cascade resolver (`resolveSidebarTarget`) decides the explorer root from a folder (workspace root > git root > filesystem). One imperative action in `App.tsx` (`focusSidebar`) orchestrates: resolve git repo once, set the explorer root mode/fsRoot, set the per-workspace git root for Source Control, and emit a reveal request. F4 wraps it (opens panel, may switch tab); autofocus wraps it silently. The previous reactive effect that tied the git root to the active terminal cwd is removed.

**Tech Stack:** React 19 + TypeScript, Vitest, Tauri IPC (`git_resolve_repo`), hugeicons, xterm OSC 7.

```ts
export function resolveSidebarTarget(input: {
  folder: string;
  workspaceRoot: string | null;
  gitRoot: string | null;
  currentFsRoot: string | null;
  home: string | null;
}): { mode: ExplorerRootMode; fsRoot: string | null } {
  const folder = normalizeSep(input.folder);
  if (input.workspaceRoot && isUnder(folder, normalizeSep(input.workspaceRoot))) {
    return { mode: "pinned", fsRoot: null };
  }
  if (input.gitRoot && isUnder(folder, normalizeSep(input.gitRoot))) {
    return { mode: "filesystem", fsRoot: normalizeSep(input.gitRoot) };
  }
  const fsRef = input.currentFsRoot ?? input.home;
  const ca = fsRef ? commonAncestor(fsRef, folder) : null;
  return { mode: "filesystem", fsRoot: ca ?? pathDirname(folder) };
}

export function migrateExplorerRootMode(
  mode: string | undefined,
): ExplorerRootMode | undefined {
  if (mode === "terminal" || mode === "git") return "filesystem";
  if (mode === "filesystem" || mode === "pinned") return mode;
  return undefined;
}
```
