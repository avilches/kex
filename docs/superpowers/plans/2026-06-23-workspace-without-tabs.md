# Workspace Without Tabs Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Closing the last tab leaves the workspace alive and empty (showing a welcome screen) instead of destroying it; a workspace is only closed by an explicit, optionally-confirmed `Cmd+W` (or the sidebar close button).

**Architecture:** The functional core (`useWorkspaces`) stops removing workspaces when the last panel of the only pane closes; it empties the pane instead. A new pure `applyClosePanel` carries the logic so it is unit-testable. Closing a workspace stays the existing `closeWorkspace`/`handleCloseWorkspace` path, now funneled through a single `requestCloseWorkspace` that shows an `AlertDialog` confirmation gated by a new `warnOnCloseWorkspace` preference. The empty pane renders a new `EmptyPaneWelcome` with action shortcuts.

**Tech Stack:** React 19 + TypeScript, Vitest, Tauri 2, shadcn/ui (AlertDialog, Switch, Checkbox), hugeicons, Tailwind v4.

- [ ] **Step 3: Add the pure `applyClosePanel` function**

In `src/modules/workspaces/lib/useWorkspaces.ts`, add this exported function next to the other `apply*` helpers (after `applyFsRoot`, around line 70). It mirrors the current `closePanel` logic but never drops a workspace:

```ts
export function applyClosePanel(
  workspaces: Workspace[],
  workspaceId: string,
  panelId: string,
): Workspace[] {
  return workspaces.map((w): Workspace => {
    if (w.id !== workspaceId) return w;
    const result = findPanelPane(w.paneTree, panelId);
    if (!result) return w;
    const { pane } = result;
    const remaining = pane.panels.filter((p) => p.id !== panelId);
    if (remaining.length === 0) {
      // Last panel in the pane. Collapse the pane unless it is the only one,
      // in which case keep the workspace alive with an empty pane.
      const newTree = removePaneFromTree(w.paneTree, pane.id);
      if (!newTree) {
        return {
          ...w,
          paneTree: updatePane(w.paneTree, pane.id, (p) => ({
            ...p,
            panels: [],
            activePanelId: null,
          })),
        };
      }
      const sibling = siblingPane(w.paneTree, pane.id);
      return {
        ...w,
        paneTree: newTree,
        activePaneId:
          w.activePaneId === pane.id
            ? (sibling?.id ?? firstPaneId(newTree))
            : w.activePaneId,
      };
    }
    const idx = pane.panels.findIndex((p) => p.id === panelId);
    const newActiveId =
      pane.activePanelId === panelId
        ? ((remaining[idx] ?? remaining[idx - 1])?.id ?? null)
        : pane.activePanelId;
    return {
      ...w,
      paneTree: updatePane(w.paneTree, pane.id, (p) => ({
        ...p,
        panels: remaining,
        activePanelId: newActiveId,
      })),
    };
  });
}
```
