# Running-Process Warning on Workspace Close Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** When closing a workspace (sidebar X or Cmd+W on an empty workspace), if the running-process warning is enabled and the workspace has terminals with foreground processes, show a single dialog listing them before killing them; otherwise fall back to the existing workspace-close confirmation.

**Architecture:** A new pure async function `collectRunningTerminals` gathers the workspace's terminals that have a foreground process (reusing the same `leafHasForegroundProcess` check as the per-tab close guard). `requestCloseWorkspace` in App.tsx becomes async and gains a three-branch cascade: process dialog, then workspace-confirm dialog, then direct close. A new `AlertDialog` in `CloseDialogs.tsx` renders the process list.

**Tech Stack:** React 19 + TypeScript, Vitest, Tauri 2, shadcn/ui (AlertDialog, Checkbox), Tailwind v4.

```ts
export async function collectRunningTerminals(
  ws: Workspace,
  getForegroundProcess: (panelId: string) => Promise<string | null>,
  getCommand: (panelId: string) => string | undefined,
): Promise<{ panelId: string; label: string }[]> {
  const terminals = allPanes(ws.paneTree)
    .flatMap((p) => p.panels)
    .filter((panel): panel is Extract<Panel, { kind: "terminal" }> => panel.kind === "terminal");
  const checked = await Promise.all(
    terminals.map(async (panel) => {
      const processName = await getForegroundProcess(panel.id);
      if (processName === null) return null;
      const label = getCommand(panel.id) ?? (processName || panel.title || "shell");
      return { panelId: panel.id, label };
    }),
  );
  return checked.filter(
    (x): x is { panelId: string; label: string } => x !== null,
  );
}
```

Replace it with:

```ts
  const [pendingCloseWorkspace, setPendingCloseWorkspace] = useState<
    { id: string; isLast: boolean } | null
  >(null);
  const [pendingWorkspaceProcesses, setPendingWorkspaceProcesses] = useState<
    { id: string; processes: { panelId: string; label: string }[] } | null
  >(null);

  const requestCloseWorkspace = useCallback(async (wsId: string) => {
    const prefs = usePreferencesStore.getState();
    const ws = workspacesRef.current.find((w) => w.id === wsId);
    if (prefs.warnOnCloseTabWithRunningProcess && ws) {
      const processes = await collectRunningTerminals(
        ws,
        leafHasForegroundProcess,
        (id) => getRunningCommandsSnapshot().get(id),
      );
      if (processes.length > 0) {
        setPendingWorkspaceProcesses({ id: wsId, processes });
        return;
      }
    }
    if (prefs.warnOnCloseWorkspace) {
      setPendingCloseWorkspace({
        id: wsId,
        isLast: workspacesRef.current.length === 1,
      });
      return;
    }
    void handleCloseWorkspaceRef.current(wsId);
  }, []);
```
